## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8 یا بالاتر
- توکن ربات تلگرام (از @BotFather بگیر)

### مراحل نصب

**1. کلون کردن پروژه**
```bash
git clone https://github.com/yourusername/telegram-group-bot.git
cd telegram-group-bot
```

**2. نصب وابستگی‌ها**
```bash
pip install -r requirements.txt
```

**3. تنظیم توکن و آیدی ادمین**
فایل `bot.py` رو باز کن و این دو خط رو تنظیم کن:
```python
TOKEN = "your_bot_token_here"  # توکن رباتت رو بذار
ADMIN_IDS = [123456789]  # آیدی عددی خودت رو بذار
```

**4. اجرای ربات**
```bash
python bot.py
```
